# Security

This profile repository contains public profile copy, links, and static assets. It should not contain credentials, private logs, browser profiles, cookies, private repository links, or internal infrastructure details.

Please do not open a public issue that includes sensitive material. If a profile link, asset, or document appears to expose private information, contact the maintainer through the public GitHub profile with a minimal report and no secrets.

If a real secret is exposed, rotate or revoke it before discussing remediation publicly.
