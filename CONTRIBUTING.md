# Contributing

Thanks for helping improve this profile repository.

## What belongs here

This repository is the public front door for Aolinge's GitHub presence. Good
contributions usually improve one of these areas:

- README clarity, wording, broken links, or project positioning
- Project cards, release links, and public navigation
- Community files such as issue templates or contribution docs
- Small visual or asset improvements used by the profile README

## Before you open an issue

Please check:

- The suggestion is relevant to this public profile repo
- The linked project, release, or URL is public and stable
- The change does not require private credentials, hidden notes, or local-only
  files

## Before you open a pull request

Please keep changes small and focused:

- One topic per pull request
- Prefer Markdown, assets, and community-file improvements
- Do not add secrets, tracking IDs, private screenshots, or unpublished links
- Update README copy carefully and keep links explicit

## Review expectations

When possible, include:

- A short summary of what changed
- Why the change helps visitors or contributors
- Screenshots only if the visual result matters

## Things to avoid

Please do not:

- Add generated noise or placeholder badges
- Add private roadmap content or internal workflow details
- Rewrite the whole profile for a small wording issue
- Introduce heavy automation or external dependencies just for profile content
